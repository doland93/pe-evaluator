# PE 집계기 — Peer Evaluation Processor

CIV 수업 동료평가(PDF/DOCX) 파일을 자동으로 읽어 엑셀로 집계하는 도구입니다.

## EXE 다운로드

1. 상단 **Actions** 탭 클릭
2. 가장 최근 실행 항목 클릭
3. 하단 **Artifacts** 에서 `PE집계기_Windows` 다운로드
4. 압축 해제 후 `PE집계기.exe` 실행

## 기능
- PDF / DOCX 동료평가 파일 자동 파싱
- 팀 번호 오기입 자동 감지 및 수정
- 개인별 100점 방식 오기입 경고
- 팀별 점수 합계 및 Weight 엑셀 자동 생성
